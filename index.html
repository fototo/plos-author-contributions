PLOS Author Contributions

This was the result of a Hack4ac team working to mine article data from the PLOS Search API and PeerJ for detailed author contributions.

Several aspects were developed simultaneously:

  • A PeerJ integration to pull author contribution information;
  • A PLOS API bridge that queries the PLOS API, enriches it with author contribution information and converts it to CSV on the fly;
  • An purely R-based implementation of mining PLOS and subsequent analysis. This project is described in more detailed here.
